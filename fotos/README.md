# Fotos do casal

Suba aqui a foto principal do casal com este nome exato:

```text
casal.jpg
```

Quando esse arquivo existir, a pagina usara essa foto automaticamente na capa e na secao de historia.

Recomendacao:

- Use uma foto horizontal ou levemente vertical, com boa luz.
- Evite fotos muito pequenas. O ideal e pelo menos `1600px` de largura.
- Se precisar trocar depois, substitua o arquivo mantendo o mesmo nome: `casal.jpg`.
